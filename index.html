Прокси для майнеров (ASIC, GPU, CPU)


В последнее время участились случаи ограничений на подключения к пулам со стороны провайдеров, в результате чего принимаемый хэшрейт на стороне пула падает.

stratum-proxy.ru представляет собой socks5 прокси для обеспечения доступа к майнинг пулам.


Доступно 2 прокси сервера размещенные в разных локациях и c разными маршрутами до майнинг пулов:

  • url: srv1.stratum-proxy.ru port: 13155

  • url: srv2.stratum-proxy.ru port: 13155

Выбирайте прокси сервер с наименьшим временем доступа.


Добавление пулов происходит вручную по запросу.


На данный момент поддерживаются следующие пулы:

  • viabtc.com

  • zpool.ca

Настройка подключения через прокси осуществляется через параметры командной строки GPU или CPU майнера.

Например, для майнера miniZ необходимо добавить следующий параметр командной строки:

--socks5=srv1.stratum-proxy.ru:13155


для майнера lolMiner:

--socks5 srv1.stratum-proxy.ru:13155


для майнеров CryptoDredge, Gminer, Rigel, SRBminer:

--proxy srv1.stratum-proxy.ru:13155


Для ASIC майнеров подключение настраивается в соответствующей настройке WEB интерфейса Вашего ASIC.


Сервис предоставляется на условиях «как есть» (AS IS).
